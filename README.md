# E-commerce-website
E-commerce website(Amazon clone) using HTML5, CSS3, Javascript, Node.js, Express.js, MongoDB, Mongoose, EJS

Run localhost:5000/

### NPM COMMANDS

- Run npm install

### INSTALL MONGODB

- Run use db e-commerce

### INSTALL elasticsearch

- Run elasticsearch
